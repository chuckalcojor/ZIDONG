# Kit de Contexto para Agentes

Este kit te ayuda a mantener **una sola fuente de verdad** para instrucciones de IA en tus proyectos.

- Archivo canonico recomendado: `AGENTS.md`
- Archivos adaptadores: `CLAUDE.md`, `GEMINI.md`, `COPILOT.md`, `CURSOR.md`, etc.
- Regla practica: actualiza primero `AGENTS.md` y luego sincroniza los demas formatos.

Contenido:

- `GUIA-AGENTS-FORMATOS.md`: explicacion completa de como funciona y como adaptarlo.
- `templates/`: plantillas listas para copiar por herramienta.
- `skill/SKILL.md`: skill para que un agente genere y mantenga estos archivos en nuevos repos.
