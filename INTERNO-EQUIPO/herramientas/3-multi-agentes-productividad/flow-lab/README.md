# Flow Lab (interno)

Pizarra local para alinear el flujo del agente conversacional antes de implementarlo en codigo.

## Uso rapido

1. Abrir `index.html` en el navegador.
2. Arrastrar tarjetas para ordenar etapas.
3. Editar objetivo y respuestas esperadas por etapa.
4. Agregar transiciones (`desde`, `hacia`, `disparador`, `etiqueta`).
5. Usar `Generar borrador` para producir texto de implementacion.

## Exportar / importar

- `Exportar JSON`: descarga el estado de la pizarra.
- `Importar JSON`: recupera una version guardada.
- Los cambios tambien quedan en `localStorage` del navegador.

## Ubicacion y alcance

- Ruta: `INTERNO-EQUIPO/herramientas/3-multi-agentes-productividad/flow-lab/`
- Es una herramienta interna de desarrollo, no parte del entregable del cliente.

## Eliminar al finalizar

Si quieren retirarlo al cerrar desarrollo, borrar esta carpeta completa:

`INTERNO-EQUIPO/herramientas/3-multi-agentes-productividad/flow-lab/`
